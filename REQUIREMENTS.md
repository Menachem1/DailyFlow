# DailyFlow – מסמך דרישות (PRD)

## חזון המוצר

אפליקציית ניהול יומי אישית המאפשרת מעקב אחר משימות קבועות ומשימות חד-פעמיות, יחד עם יומן יומי לסיכום היום. מיועדת לשימוש אישי, פשוטה ומהירה לשימוש.

---

## שלב 1 – MVP (הפיתוח הנוכחי)

- Frontend בלבד (Vue 3)
- כל הנתונים נשמרים ב-`localStorage`
- ללא backend, ללא משתמשים, ללא התחברות

---

## מודולים

### 1. משימות קבועות (Recurring Tasks)

משימות שחוזרות על עצמן לפי לוח זמנים קבוע.

**תכונות:**
- יצירת משימה קבועה עם שם וסדר עדיפויות
- הגדרת תדירות:
  - כל יום (7/7)
  - ימים ספציפיים בשבוע (לדוגמה: א׳–ה׳, 5 פעמים בשבוע)
- כל יום מוצגת רשימת המשימות הרלוונטיות לאותו יום
- סימון ✅ / ביטול סימון של כל משימה לאותו יום
- היסטוריה – מעקב אחר ביצוע לפי תאריך (כמה ימים בוצעה המשימה)

**דוגמאות למשימות קבועות:**
- תפילת שחרית – כל יום
- תפילת מנחה – כל יום
- תפילת ערבית – כל יום
- חברותה – א׳–ה׳
- אימון גופני – א׳, ג׳, ה׳

---

### 2. משימות חד-פעמיות (One-Time Tasks)

משימות אד-הוק שצריך לבצע בנקודת זמן מסוימת.

**תכונות:**
- הוספת משימה עם: כותרת, תיאור (אופציונלי), תאריך יעד (אופציונלי)
- סימון כ"בוצע" וארכוב אוטומטי
- מחיקת משימה
- תצוגת משימות פתוחות vs. הושלמו

**דוגמאות:**
- לסדר עניין בבנק
- תור לרופא
- לשלוח מייל ל...

---

### 3. יומן יומי (Daily Journal)

מרחב לכתיבת סיכום יומי קצר.

**תכונות:**
- כתיבת רשומה יומית חופשית (כמה משפטים)
- כל רשומה משויכת לתאריך
- צפייה ברשומות של ימים קודמים (לפי תאריך)
- עריכה של רשומת היום הנוכחי

---

### 4. הקלטת קול ליצירת משימות (Voice-to-Task)

**תכונות:**
- לחצן הקלטה בממשק
- הקלטת קול דרך המיקרופון
- המרה לטקסט (Web Speech API)
- שליחת הטקסט למנוע AI (Claude API) ליצירת משימות מובנות אוטומטית
- תצוגת המשימות שנוצרו לאישור המשתמש לפני שמירה

> **הערה:** פיצ'ר זה יפותח בשלב 1 – נחשב לחלק מה-MVP בשל חשיבותו.

## מבנה הנתונים ב-localStorage

```json
// משימות קבועות
"recurring_tasks": [
  {
    "id": "uuid",
    "title": "תפילת שחרית",
    "days": [0,1,2,3,4,5,6],
    "createdAt": "2026-04-15"
  }
]

// ביצועים יומיים
"daily_completions": {
  "2026-04-15": {
    "recurring": ["task-id-1", "task-id-2"],
    "oneTime": ["task-id-3"]
  }
}

// משימות חד-פעמיות
"one_time_tasks": [
  {
    "id": "uuid",
    "title": "לסדר בבנק",
    "description": "",
    "dueDate": "2026-04-20",
    "completed": false,
    "createdAt": "2026-04-15"
  }
]

// יומן
"journal": {
  "2026-04-15": "היום היה יום טוב, בוצעו כל המשימות..."
}
```

## מסכים (Views)

| מסך | תיאור |
|-----|--------|
| **Today** | מסך הבית – משימות קבועות ליום זה + משימות חד-פעמיות פתוחות |
| **Tasks** | ניהול כל המשימות (קבועות + חד-פעמיות) |
| **Journal** | יומן יומי – כתיבה וצפייה |
| **History** | סטטיסטיקות וביצועים לאורך זמן |

## טכנולוגיות

| שכבה | טכנולוגיה |
|------|-----------|
| Framework | Vue 3 + Vite |
| State | `localStorage` (ללא Pinia בשלב זה) |
| Styling | CSS / TailwindCSS |
| Voice | Web Speech API |
| AI | Claude API (לעיבוד הקלטה למשימות) |
